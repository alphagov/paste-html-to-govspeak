This document serves as an example of HTML elements that should be converted to Govspeak when the copy of this document is copied into the example. It is intended to demonstrate a best case of the pasting experience.

Unordered lists are supported:

  • an item
  • another item

As are ordered lists:

  1. Item 1.
  2. Item 2.

And nested lists:

  • an item with sub items:
    • item 1
    • item 2
  • another item
Preformatted code is also supported
